# Simple-FreeRTOS-Project

Software used: Microchip MPLab Harmony V2, FreeRTOS, PuTTY

Hardware used: PIC32MX470F512L development board, RS-232 Interface Cable, LIS3DH Triple-Axis Accelerometer

Must Import to Harmony Project to run. 

Using SPI, The PIC32 communicates with a PuTTY terminal and the Accelerometer to display the axis values.
Buttons are pressed to display either X axis or Y axis values.
